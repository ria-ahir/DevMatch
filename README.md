# DevMatch

**Find your people. Build your project.**

DevMatch is a student-only platform that helps students find collaborators, teammates, and project opportunities. It connects students with complementary skills so they can turn ideas into real projects.

Many students have ideas but struggle to find the right team. Others have skills but no project to contribute to. DevMatch solves this by helping students discover projects, connect with collaborators, and build together.

---

# The Problem

Students frequently face these challenges:

• “I have an idea but need a designer / frontend developer / marketer.”  
• “I have skills but no project to join.”  
• “I want to build something but none of my friends are interested.”  
• “I want to meet ambitious students outside my program.”  
• “Public platforms are too broad — I want to work with other students.”

Despite the strong interest in collaboration, there is no platform specifically designed for students to find project partners.

In fact, studies show that **96% of developers report enjoying programming more when collaborating with others**.

---

# The Solution

**DevMatch** is a platform where students can:

• Post projects and recruit collaborators  
• Discover projects they can contribute to  
• Match with students who have complementary skills  
• Connect and message potential teammates  
• Build a portfolio of projects and collaborations

The platform is designed specifically for students, starting with universities like the **University of Alberta**, with potential verification through student accounts or OneCard.

---

# Key Features

### Project Discovery
Browse projects created by other students and find opportunities to contribute.

### Post a Project
Students can create projects and specify:
- required skills
- project stage
- time commitment
- collaborators needed

### Collaborator Matching
DevMatch helps students discover other students with compatible:
- skills
- interests
- goals

### Messaging
Students can connect and communicate with potential teammates.

### Student Profiles
Each user can build a profile including:
- skills
- interests
- projects
- collaboration goals

# How We Built It

We built DevMatch as a mobile application using **React Native**.

Our development workflow included:

- Designing the UI and flows in **Figma**
- Building reusable components for cards, tags, and UI elements
- Implementing core screens including project browsing, matching, and messaging
- Structuring the app with scalable components and navigation

We divided responsibilities across team members so each person owned a major part of the product:

- Home + App Architecture
- Projects System
- Match + Messaging
- Profile + Authentication
